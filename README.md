# Voice Recorder Python Script 🎙️

This Python script allows you to **record audio from your microphone** and save it as a WAV file.  
The user can manually stop the recording using **Ctrl+C**.

---

## Features

- Record audio from your microphone.
- Manual stop (user-controlled).
- Saves audio in WAV format using `wavio`.
- Plays back the recorded audio immediately.
- Works on Mac, Windows, Linux (Python 3.10+ recommended).

---

## Installation

1. **Clone this repository:**

```bash
git clone https://github.com/mohitjaryal/voice-recorder.git
cd voice-recorder
```
## Create a virtual environment
### macOS / Linux:
```bash
python3 -m venv myenv
source myenv/bin/activate
```
### windows
```bash
python -m venv myenv
myenv\Scripts\activate
```

## Install dependencies:
```bash
pip install -r requirements.txt
```

## Run the script:
```bash
python3 main.py
```
